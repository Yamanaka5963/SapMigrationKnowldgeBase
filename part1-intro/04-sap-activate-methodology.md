# 04 - SAP Activate Methodology Overview

SAP Activate is the standard project methodology for S/4HANA migrations. It replaced the legacy ASAP methodology and follows an agile approach.

## 6 Phases

| Phase | Key Activities |
|---|---|
| **1. Discover** | Run Readiness Check, assess landscape, identify migration approach, initial scoping |
| **2. Prepare** | Set up project governance, system infrastructure, team onboarding, initial backlog |
| **3. Explore** | Fit-to-standard workshops, configure baseline, identify gaps and custom requirements |
| **4. Realize** | Build configuration, develop custom code, iterative testing (unit, integration) |
| **5. Deploy** | Cutover planning, data migration, user training, go-live execution |
| **6. Run** | Hypercare, issue resolution, handoff to operations |

> For detailed phase-by-phase execution guidance, see [Part 2 — 10: Client Engagement Intake](../part2-pro/10-client-engagement-intake.md) onwards.
